How to Create an Custom Lithophane Keychain

If you own a 3d printer and search for an christmas or birthday present, a custom lithophane Keychain is perfect for you.

Used Software

  • Image to Lithophane (3dp.rocks/lithophane)
  • Paint.net (Download)
  • Blender (Download)
  • PrusaSlicer (Download)

These Programms are needed to create an Lithophane Keychain, however you dont need to know how to use them.

Prepare Files

At first you need to choose a Picture for the Keychain. The Subject of the picture should be in the center because we will apply an "vignette effect". Download all the Model files for this project from prusaprinters: [Project Link] andall of them with youre Picture in a Folder to keep an overview.

Edit Picture

To Edit the Picture we use Paint.net. Start the Programm and open your Picture in it (Ctrl + O). For this tutorial I use the Picture of an Benchy!

At first you have to make sure that the picture has the same height and width. For that click on Image>Canvas Size. A Popup window will appear. There unselect "maintain aspect ratio" and set the width equal to height (or reverse depending on which side is longer. The Image should'nt be cropped).


Next we apply an "vignette effect". Click on effects>photo>vignette. In the Popup Window set the Center to 0;0, Radius to 0.50 and Density to 1.

Now this step is finshed and you can save the Image in your Folder.

Convert Picture to Stl

Now we convert our Picture to an Lithophane on the following page: 3dp.rocks/lithophane. Open the web page and upload your Picture ("Images" tab).
Click on Settings>Image Settings. Now check that the settings are like in the picture below.

Then click on Settings>Model Settings and check that the settings are like in the picture below.

When the Settings are set you can click on refresh in the model tab and then download it.

Convert Stl to Keychain

The stl file is now an square with your Picture on it. To transform it we use Blender. Open the Programm and delete the "Standard Cube" (Grey Cube in the Center, Delete with "Del").
Now open your Files by clicking on File>Import>Stl. In the Popup select "centercircle.stl", "keychain top.stl" and the stl you generated in the previous step. If everything is correct it should look like this:

In the next step click on your "generated" file (the file generated on 3dp.rocks/lithophane) in the top right. Then on the little Wrench below and click "Add Modifier". Select "Boolean". Repeat this so you have to modifiers.

In the first modifier Tab select "Intersect" and select "Centercircle" as Object.
In the second modifier Tab select "Union", select "Keychain Top" as Object and set the "Overlap Threshold to 0.1m.
The Modifier Tab should now look like this:

To save the file click on your "generated" File in the top right. Then go to File>Export>Stl. In the Popup select "Selection Only" and then click on "Export Stl". You now have the finished Keychain as an .stl.

Slice it

Since you own a 3d Printer you porpably know how to use a Slicer. But here are some notes:

  • Most of the Time Blenders "Boolean Operator" is faulty. So after import into the PrusaSlicer you should click on the triangle by the filename to repair it.
  • You have to Print the Keychain up right (Like in the Picture below) and therfore enable "Support material on Build plate only"
  • Print with an Layerheight of 0.1mm and 4 Perimeters

Now you are ready for printing! Have Fun!
